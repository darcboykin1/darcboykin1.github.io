Inter_EAT

An application that takes advantage of the Google Maps, Geolocation, and Yelp APIs. The data from the Google Maps and Geolocation APIs are sent to the Yelp API. The Yelp API then renders locations relative to the user's location, based on the choice choice of cuisine they selected from a prepopulated list.

  • NodeJS
  • ExpressJS
  • Google/Yelp APIs
  • Axios

Library Adventure RPG

A small game made in Javascript inspired by the old-school Final Fantasy games. The characters are two instances of one javascript object. The object has an attack and defend method. Each character has a chance land a critical hit to land more damage. The goal is to strategically use your attacks and potions do defeat the other character before he kills you.

  • Javascript ES6 Classes
  • jQuery
  • Object Oriented Programming

RemindR

This app was developed to save entertainment, separated by category, that they wish to consume at a later date.
